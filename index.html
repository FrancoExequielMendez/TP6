2) Para integrar JavaScript a HTML se usa la etiqueta "script" y se linkea con la direccion del archivo.

3) Las variables se pueden declarar usando var , const , o let; var tiene alcance de función, mientras que const y let tienen alcance de bloque. Las variables const no pueden ser reasignadas, mientras que las variables let si pueden serlo.
JavaScript tiene la característica de ser un lenguaje débilmente tipado con tipado dinámico, es decir, que todos los tipos de datos se pueden asignar o reasignar a las variables que los almacenan.
Las expresiones regulares son patrones que se utilizan para hacer coincidir combinaciones de caracteres en cadenas. En JavaScript las expresiones regulares también son objetos

4) Las formas de declarar son:
Function declaration.
Function expression.
IIFE (Immediately Invoked Function Expression).
Shorthand method definition.
Arrow function.
Generator function.
Function constructor.

5) JavaScript tiene los siguientes tipos de operadores:
Operadores de asignación.
Operadores de comparación.
Operadores aritméticos.
Operadores bit a bit.
Operadores lógicos.
Operadores de cadena.
Operador condicional (ternario).
Operador coma.
Operadores unarios.
Operadores relacionales.
Para realizar conversiones se coloca el tipo destino entre paréntesis y a la izquierda del valor que queremos convertir: Tipo VariableNueva = (NuevoTipo) VariableAntigua
